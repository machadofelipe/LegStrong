IQMATHLIB

Integer-Quotient Math (IQMATH) Module

Description

This module contains the IQ Math library.

Please see the functions define in IQmathLib.h for more details.

Integration Information

There is only one module in this package, the APIs can be referenced at IQMATHLIB. The API headers can be located at IQmathLib.h. The test procedures are described at test_iqmath.h.
