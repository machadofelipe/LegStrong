enum ST_PosConvStatus_e

Enumeration for the Position Convert Status states.

Enumerator
ST_POS_CONV_IDLE 

Position Converter is in idle state, zero output.

ST_POS_CONV_INIT 

Position Converter is in init state, validating configured parameters.
